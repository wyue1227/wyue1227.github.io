Windows搭建Gosu开发环境

背景

目前的项目基于Guidewire框架，使用的开发语言是Gosu。在个人机器上搭建Gosu的开发环境用于学习语法特性。

环境搭建

前期准备

  1. IntelliJ IDEA 2019.3.5
  2. OS Gosu插件（最新版仅支持2019.3.5，之后的版本无法导入至IDEA）
  3. Gosu运行环境（gosu-1.14.16-full.zip）
  4. JDK1.8 + （Gosu依赖基于JDK1.8 +）

配置手顺

校验JDK

在命令行中执行java -version，查看JDK版本是否 > 1.8。

下载Gosu运行环境并解压缩

https://gosu-lang.github.io/downloads.html

IDEA中导入OS Gosu插件（本地导入或在线下载）

创建一个普通的Java工程

Create New Project

2019.3.5版本的默认JDK为1.11，编译Gosu会产生doc警告，所以更正为JDK8

Finish

创建完成

导入Gosu依赖

File -> Project Structure

Libraries -> Java

选择步骤2.2.2解压缩完的文件夹下的所有依赖 jar 包
