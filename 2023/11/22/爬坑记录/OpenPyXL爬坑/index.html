图形问题解决

根据运行结果可以看出，只是简单的打开、关闭一个 excel 文件，会导致图片以及图形的丢失。

针对图形丢失在网上有很明确的说明，openpyxl 使用 Pillow 处理图片，所以需要安装 Pillow

pip install Pillow
